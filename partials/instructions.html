Instructions

Welcome to Magic December! To use the site, you can click on the cards for each day to find out about festivals or celebrations taking place on that day.

Be sure to click and hold on each card for a few seconds to view the full animated effect!

If you would like to mute the sound effect, please click the volume icon just below the page title.

If you would like to visit the GitHub Repo, please click the GitHub icon next to the volume button.
